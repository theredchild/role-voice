# Changelog

## 0.1.0 (Unreleased)

### Added
- Push-to-talk voice recording with configurable hotkey
- Local STT via mlx-whisper (Apple Silicon) and faster-whisper (CPU/CUDA)
- Silero VAD silence trimming for faster transcription
- Target dispatchers: clipboard, Claude Code CLI, terminal paste
- YAML configuration with CLI overrides
- Rich console UI with recording/transcription feedback
- Benchmark command for STT latency testing
- Audio device listing command
